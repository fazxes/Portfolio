# Portfolio

Personal portfolio site for Pranit Sharma ([@fazxes](https://x.com/fazxes)).

Built with Next.js, [shadcn/ui](https://ui.shadcn.com/), and [Magic UI](https://magicui.design/). Based on the [magicuidesign/portfolio](https://github.com/magicuidesign/portfolio) template.

## Getting Started

```bash
pnpm install
pnpm dev
```

Edit [`src/data/resume.tsx`](./src/data/resume.tsx) to update site content.
