## Selected Work

- **[Orbit](https://orbit.build/)**. AI-native IDE for macOS. One agent with full context across editor, browser, terminal, and docs. Ship from branch to PR in a single conversation.
- **[Agents SDK](https://www.agentsdk.build/)**. Open-source Rust SDK for production AI agents. Type-safe, async-first primitives, zero runtime overhead.
- **[Nightshift](https://orbit.build/nightshift)**. Autonomous overnight codebase improvement agent. Run it before bed, wake up to a reviewed worktree and shift log.
- **[OPS](https://orbit.build/ops)**. Orbit Protocol Server. 17 subsystems of real-time code quality enforcement. Deterministic rules, not prompt suggestions.
- **[X Agent](https://orbit.build/x-agent)**. Algorithm-optimized X engagement agent. Monitors, drafts replies with Phoenix scoring, posts through the browser.
- **[Skills](https://orbit.build/skills)**. Modular AI agent skills for Orbit, Claude Code, Cursor, Codex, and 38+ coding assistants.
- **[Mosaic](https://www.mosaic.sh/)**. In-app testing framework that runs inside live web apps. Zero mocks, real state, real race conditions.

Full list and case studies at [fazxes.com](https://fazxes.com).
